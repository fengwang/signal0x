\item range--based for loop
\item rvalue reference

\item ...
\end{enumerate}

\paragraph{This example writes "Hello World." using signals and slots in the following way:}
\begin{enumerate}
\item create a signal sig, a signal that takes no arguments and has a void return value 
\item connect the hello\_world function to the signal using the connect method
\item the signal is triggered, which in turns invokes hello\_world() to print "Hello World.".
\end{enumerate}
